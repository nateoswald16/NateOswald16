## 🛠️ What I Build

### [Seam Analytics](https://github.com/nateoswald16/SEAM-ANALYTICS) — Real-Time MLB Desktop App
A full-featured Windows desktop application for MLB stat analysis and live game tracking. Ships as a proper installer with auto-updates.

- **PyQt6** GUI with live scoreboards, lineup viewers, batter-vs-pitcher tables, and season leaderboards
- Dual **SQLite** databases (~780 MB) ingesting data from the MLB Stats API and Baseball Savant via `pybaseball`
- Packaged with **PyInstaller + Inno Setup**; auto-update system checks GitHub Releases on launch
- **14 versioned releases** and counting

### [Personal Portfolio Website](https://github.com/nateoswald16/Website-Portfolio) — Full-Stack Web App
A full-stack portfolio site built as my CS50x final project. Bridges server-side and client-side rendering in a single cohesive app.

- **Python/Flask** backend with **Jinja2** templating and **SQL** database for all project data
- **React** frontend components with animated background, light/dark mode toggle, and smooth scroll UX
- Secure **admin dashboard** with session-based auth and full CRUD for project management
- Accessibility-first: ARIA labels and semantic HTML throughout
